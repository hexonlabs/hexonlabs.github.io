rocket_launch What Are TGPlugs?

TGPlugs are modular extensions for TGBrowser that unlock endless possibilities:

  • Modify appearance with custom CSS skins (style.css)
  • Add functionality with JavaScript (main.js)
  • Register with TGMoLink API for enhanced integration
  • Communicate with other mods seamlessly

new_releases What's New? v26.01.07a+

Enhanced Security & Validation

  • TGUID System: All TGPlugs MUST register with a unique identifier (e.g., com.shaman.darktheme).
  • Gatekeeper Protection: Mods without valid registration trigger warnings.

Breaking Changes

CRITICAL: Old mods without window.TGMoLink.register() will trigger security warnings. Update them immediately.

folder File Structure

github repo for tgplugs/ └── YourModFolder/ (or TGUID) ├── style.css (optional, visual) └── main.js (required, logic)

code Basic Template

main.js (Required)

// TGPlug: Midnight Theme // TGUID: com.shaman.midnighttheme (function() { 'use strict'; const MOD_ID = 'com.shaman.midnighttheme'; // Check if API exists if (window.TGMoLink && window.TGMoLink.register) { window.TGMoLink.register(MOD_ID); init(); } else { // Retry logic const check = setInterval(() => { if (window.TGMoLink && window.TGMoLink.register) { clearInterval(check); window.TGMoLink.register(MOD_ID); init(); } }, 100); } function init() { console.log("Mod Loaded!"); // Your logic here } })();

verified Best Practices

1

Always Register with TGUID

Without registration, your mod is flagged as unsafe.

2

Namespace Your Code

Use IIFE (Immediately Invoked Function Expressions) to avoid polluting the global scope.

3

Security First

Never store sensitive data (passwords, tokens) in localStorage.
